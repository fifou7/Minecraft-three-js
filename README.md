# Biome minecraft three js

1. Cloner le Repository
Accéder au Repository GitHub

Rendez-vous sur la page GitHub du projet.

Cloner le projet en local

Copiez l'URL du repository.

Ouvrez votre terminal et exécutez :

git clone https://github.com/username/nom-du-projet.git

Remplacez https://github.com/username/nom-du-projet.git par l'URL de votre repository.

2. Naviguer dans le Répertoire du Projet
   
Déplacez-vous dans le répertoire du projet :

cd nom-du-projet

3. Vérifier les Pré-requis

Node.js 

Vite

Three.js
Pour vérifier: 

git --version
node -v
npm -v

4. Installer les Dépendances avec : npm install

Cela installera automatiquement Vite, Three.js, et les autres modules nécessaires si définis dans package.json.

5. Lancer le Projet en Local

6. Dans le dossier JavaScript, faites CRTL J pour ouvrir le terminal
   
7. Utilisez Vite pour démarrer le serveur en tapant dans la console: npm run dev

L'URL par défaut est généralement http://localhost:5173.

Le projet est lancé
